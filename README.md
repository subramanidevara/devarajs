QR-BASED-STUDENT-VERIFICATION
QR Codebased student verification and attendance management system built with Python and Flask. It automates identity verification, reduces manual effort, and includes machine learningbased attendance analysis
Technology Stack
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Database: SQLite or MySQL
Machine Learning: Pandas, NumPy, Scikit-learn
Visualization: Chart.js
Installation Guide
Prerequisites
Python 3.8 or higher
pip package manager
Steps
Clone the repository
git clone https://github.com/your-username/your-repo-name.git
Navigate to the project directory
cd your-repo-name
Install dependencies
pip install flask qrcode reportlab pandas numpy scikit-learn
Run the application
python app.py
Open in browser
http://127.0.0.1:5000
Machine Learning Overview
Proxy attendance detection using frequency-based pattern analysis
Anomaly detection for irregular attendance behavior
Attendance risk prediction based on historical data
